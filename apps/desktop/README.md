# Desktop App Area

This directory is reserved for product-specific code and integration notes.

The short-term plan is:

1. validate the upstream `openshot-qt` app in `upstream/openshot-qt`
2. identify the minimum set of product changes needed for retiming and transitions
3. move custom UI, presets, and helper code into this workspace area where practical

Until the baseline is running, most app-level exploration will happen in the upstream OpenShot repositories.
